# Algorithmic Data Structures in C

## What this project is

This project is a collection of fundamental data structures implemented from scratch in the C programming language. Instead of relying on existing libraries, every data structure is built manually to understand how memory, pointers, and algorithms work at a low level.

The goal of this project is to strengthen the foundations required for systems programming, algorithm design, and future work in high-performance computing and machine learning systems.

---

## What I built so far

Currently, this project includes:

- Singly Linked List
  - Insertion
  - Deletion
  - Traversal
  - Search

- Doubly Linked List
  - Forward and backward traversal
  - Insertion
  - Deletion
  - Search

- Binary Search Tree (BST)
  - Node insertion
  - Node deletion
  - Search
  - Inorder traversal
  - Preorder traversal
  - Postorder traversal

Additional data structures and algorithms will be added as the project grows.

---

## Why I built it in C

I chose C because it exposes the underlying mechanics that higher-level languages abstract away.

Implementing these data structures in C requires manual memory management, explicit pointer manipulation, and careful control over data layout. Building them from scratch provides a much deeper understanding of how dynamic memory allocation and algorithmic operations work internally.

Rather than using library implementations, this project focuses on understanding the computational foundations that underpin operating systems, compilers, databases, and modern machine learning frameworks.
